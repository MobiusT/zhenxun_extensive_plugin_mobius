# traditional2simplified

繁中转简中
想要机器人发送的内容

## 使用

簡體/簡體字/簡中+想要轉換的文字     #以对应的简体字执行命令

輸出簡體/簡體字/簡中+想要轉換的文字  #回复转换结果

更新映射  #更新map.json映射关系

在插件目录下新建map.json写入转换关系：{"需要调整的字": "解析出错的结果需要显示的结果"}  

例{"德": "得的"， "斯": "死四"}       

原文为 德斯  且转换结果为 得死 时，返回 的四
